02

Energy & Power signals

Energy signal

Total energyE = ∫₋∞^∞ |x(t)|² dt
0 < E < ∞. Then P = 0. Signal "dies out".

A decaying exponential e^(−at)u(t), a = 0 is an energy signal. Finite energy means the signal eventually becomes negligible.

Power signal

Average powerP = lim(T→∞) (1/2T) ∫₋ᵀᵀ |x(t)|² dt
0 < P < ∞. Then E = ∞.

Any periodic signal is always a power signal. It persists forever — infinite total energy, but finite average power per unit time.

⚠ Three rules — always tested
  • A signal cannot be both energy and power.
  • A signal can be neither — e.g. x(t) = t has infinite energy AND infinite power.
  • Any periodic signal is always a power signal (infinite energy).
03

Even & Odd signals

Even: x_e(−t) = x_e(t)

Mirror symmetry about the y-axis. Always zero derivative at t = 0. Examples: cos(t), t², |t|, any constant.

Odd: x_o(−t) = −x_o(t)

Antisymmetric — fold over y-axis and negate. Must satisfy x_o(0) = 0. Examples: sin(t), t, t³.

Even/Odd decomposition — works for ANY signal

Even partx_e(t) = ½[x(t) + x(−t)]
Odd partx_o(t) = ½[x(t) − x(−t)]

Then: x(t) = x_e(t) + x_o(t). This always works — any signal has a unique even/odd decomposition.

Six properties of even/odd functions (Q4 in both exam papers!)

  • Sum of two even = even
  • Sum of two odd = odd
  • Sum of even + odd = neither
  • Product of two even = even
  • Product of two odd = even (negatives cancel!)
  • Product of even × odd = odd

Key integral: ∫₋∞^∞ x_e(t)·x_o(t) dt = 0 always. The product of even × odd is odd, and the integral of any odd function over a symmetric interval is zero.

04

Periodic signals

Definition

x(t) = x(t + T₀), T₀ > 0 (smallest such T₀)

Also valid: ω₀ = 2π/T₀ (fundamental angular frequency). A non-periodic signal is aperiodic.

Sum of periodic signals — when is it periodic?

The sum x₁(t) + x₂(t) is periodic if and only if T₁/T₂ is a rational number (ratio of integers).

To find T₀: express T₁/T₂ = p/q in lowest terms. Then T₀ = q·T₁ = p·T₂.

05

Singularity functions & Impulse

Unit step u(t)

u(t) = 1 for t ≥ 0, 0 for t < 0

Switches on at t = 0. u(t−a) switches on at t = a. Used to "gate" signals.

Ramp r(t)

r(t) = t·u(t) = ∫u(t)dt

Linearly increasing from t = 0. Derivative of ramp = step. Integral of step = ramp.

Impulse δ(t)

δ(t) = 0 for t ≠ 0, and ∫₋∞^∞ δ(t) dt = 1

Infinite height, zero width, unit area. Physically represents a very brief, intense pulse.

Sifting property (most used!)

∫₋∞^∞ f(t)·δ(t − a) dt = f(a)

The impulse "sifts out" the value of f at t = a. Also: f(t)·δ(t−a) = f(a)·δ(t−a).

Key relationship: δ(t) = du(t)/dt and u(t) = ∫δ(τ)dτ. The impulse is the "derivative" of the step function — even though the step isn't classically differentiable.

⚠ Common mistakes in Topic 1
  • Using round brackets x(t) for discrete — always use x[n] with square brackets
  • Forgetting x_o(0) = 0 must hold for odd signals
  • Assuming sum of periodic signals is periodic without checking T₁/T₂ is rational
  • Any signal involving π in the period ratio is likely irrational → not periodic

Signal Transformations

Time reversal, time shifting, time scaling, and combined transformations — with step-by-step interactive diagrams. This is the most commonly confused area, so read carefully.

01

The Golden Rule for combined transforms

How to handle x(at + b) — always do this order

Write it as x(a(t + b/a))

Factor out the coefficient of t first so you can see the shift clearly.

Apply time shift first: t → t + b/a

Shift the signal left by b/a (if b/a > 0) or right by |b/a| (if b/a < 0).

Then apply scaling: t → at

If |a| > 1: compress. If 0 < |a| < 1: stretch. If a < 0: also reflect.

Alternatively: find where the new signal equals the originalNew signal starts where a·t + b = original start, ends where a·t + b = original end
02

Time Reversal: x(t) → x(−t)

What it does

Replace every t with −t. The signal is flipped/reflected about the vertical axis (t = 0). Think of it as the mirror image of the original. What was on the right is now on the left, and vice versa.

y(t) = x(−t)

Worked example: Reverse x(t)

Original x(t): a ramp from t=0 to t=2 (value goes from 0 to 1), then flat at 1 for t=2 to t=4.

Key insight: the ramp that was going up from 0 to 2 now appears going down from −2 to 0. The flat section that was from 2 to 4 is now from −4 to −2. Everything mirrors about t = 0.

03

Time Shifting: x(t) → x(t − t₀)

The rule — easily confused!

x(t − t₀): shift RIGHT by t₀ if t₀ > 0 (delay)
x(t + t₀): shift LEFT by t₀ if t₀ > 0 (advance)

Memory trick: x(t − 3) — set t − 3 = 0, so t = 3. The signal that was at 0 is now at t = 3. It shifted RIGHT. x(t + 3) — set t + 3 = 0, so t = −3. The signal that was at 0 is now at t = −3. It shifted LEFT.

Worked example: x(t) → x(t − 2) (delay by 2)

04

Time Scaling: x(t) → x(at)

Rules

|a| > 1: signal COMPRESSES (speeds up, shorter duration)
0 < |a| < 1: signal STRETCHES (slows down, longer duration)
a < 0: scaling also REFLECTS about t = 0

How to find new endpoints: if original x goes from t=a to t=b, then x(2t) goes from t=a/2 to t=b/2. x(t/2) goes from t=2a to t=2b. Divide all time values by the scale factor.

Worked example: x(t) → x(2t) (compress by 2)

05

Combined: x(at + b)

Textbook Example 2.1 from Dr. Khoo's W1 slides

Problem: Given signal x(τ) that is: flat at −1 for t∈[−1, 0), ramp from 0 to 1 for t∈[0,1], flat at 1 for t∈[1,2]. Find y(t) = x(1 − t/2).

Rewrite: x(1 − t/2) = x(−(1/2)(t − 2))

Factor: a = −1/2, b = 1. So we have reflection + scaling by ½ + shift by 2.

Find new endpoints by substitution

Original starts at τ = −1: set 1 − t/2 = −1 → t = 4. Original ends at τ = 2: set 1 − t/2 = 2 → t = −2. So x(1−t/2) runs from t = −2 to t = 4.

Note: a = −1/2 < 0, so signal is also reflected

The ramp that was rising (τ going from 0 to 1) now falls (t going from 2 to 0). The flat −1 region maps to t ∈ [2, 4].

Stretching by factor 2 (|a|=0.5 → stretch)

The original width 3 becomes width 6. Runs from t=−2 to t=4. This matches Figure 2.4 from the textbook.

Quick-reference: what each parameter does to x(at + b)

  • b > 0 (e.g. x(t+3)): shift LEFT by b — signal appears earlier
  • b < 0 (e.g. x(t−3)): shift RIGHT by |b| — signal appears later (delayed)
  • a > 1 (e.g. x(2t)): compress — signal is shorter, same height
  • 0 < a < 1 (e.g. x(t/2)): stretch — signal is longer, same height
  • a < 0 (e.g. x(−t)): reflect about t = 0, then scale by |a|
  • Amplitude scale A (e.g. 3x(t)): multiply all vertical values by 3, no time change
  • DC shift B (e.g. x(t)+B): raise entire signal up/down, no shape change
06

Building signals from steps and ramps

Representing piecewise signals using u(t) and r(t)

Any piecewise-linear signal can be expressed as a sum of steps and ramps. The key is to track slope changes at each breakpoint.

Rectangular pulse (from lecture slide 18)rect(t) = u(t) − u(t−1)
From lecture slide 19: signal if(t) = r(t) − r(t−1)ramp from 0→1, then flat — slope of ramp cancels at t=1
From lecture slide 19: signal iix(t) = −r(t+1) + 2r(t) − r(t−2) − u(t−3)V-shape peaking at t=0, flat then step down

Procedure for writing a piecewise signal

Identify all breakpoints (where slope or value changes)

Look at every corner and jump in the signal graph.

At each breakpoint tₖ, determine the slope change Δm

Add Δm · r(t − tₖ) to the expression. This adjusts the slope from tₖ onward.

At each jump discontinuity, add a step

A jump of amplitude A at tₖ adds A · u(t − tₖ).

Check: differentiate your expression and verify slopes match

d/dt of your result should give the correct piecewise derivative of the signal.

⚠ The most common exam mistakes with transformations
  • x(t − 2): students say "shift left" — it's a shift RIGHT (delay). Set t−2=0 → t=2, the signal moves to the right.
  • x(2t): students say "stretch" — it's a COMPRESS. Higher argument = faster → shorter signal.
  • x(1 − t/2): don't forget the reflection! The negative sign on t reverses the signal.
  • Amplitude vs time: x(t) + 3 shifts the signal UP; x(t + 3) shifts the signal LEFT. They look similar on paper but are completely different.

Worked Examples

Step-by-step solutions to the type of signal questions that appear in the exam — energy/power classification, even/odd decomposition, and periodicity.

EX1

Classify x(t) = e^(−3t)u(t) as energy, power, or neither

Compute total energy E

E = ∫₀^∞ |e^(−3t)|² dt = ∫₀^∞ e^(−6t) dt = [−e^(−6t)/6]₀^∞ = 0 − (−1/6) = 1/6

Check: 0 < E < ∞?

E = 1/6. Yes — finite and positive.

Conclude

Energy signal. P = 0. The signal decays to zero, so its average power over all time is zero.

EX2

Find the even and odd parts of x(t) = u(t) (unit step)

Find x(−t)

x(−t) = u(−t) = 1 for t ≤ 0, 0 for t > 0. This is a unit step flipped about the origin.

Compute x_e(t) = ½[x(t) + x(−t)]

For t > 0: ½[1 + 0] = 1/2. For t < 0: ½[0 + 1] = 1/2. At t = 0: ½[1 + 1] = 1/2 (using u(0) = 1 convention).

x_e(t) = 1/2 for all t ≠ 0 — a constant DC signal (even ✓).

Compute x_o(t) = ½[x(t) − x(−t)]

For t > 0: ½[1 − 0] = 1/2. For t < 0: ½[0 − 1] = −1/2. At t = 0: 0.

x_o(t) = ½ sgn(t) — the signum function scaled by ½ (odd ✓, since x_o(0) = 0).

Verify: x_e(t) + x_o(t) = x(t)

For t > 0: 1/2 + 1/2 = 1 = u(t) ✓. For t < 0: 1/2 − 1/2 = 0 = u(t) ✓.

EX3

Is x(t) = cos(2t) + sin(3t) periodic? Find T₀.

Find individual periods

cos(2t): ω₁ = 2 → T₁ = 2π/2 = π. sin(3t): ω₂ = 3 → T₂ = 2π/3.

Check T₁/T₂

T₁/T₂ = π ÷ (2π/3) = π × 3/(2π) = 3/2. Rational number → periodic! ✓

Find T₀ = LCM(T₁, T₂)

T₀ = LCM(π, 2π/3). Express as multiples: π = 3 × (π/3), 2π/3 = 2 × (π/3). LCM = 6 × (π/3) = 2π.

Verify: cos(2(t+2π)) = cos(2t+4π) = cos(2t) ✓. sin(3(t+2π)) = sin(3t+6π) = sin(3t) ✓.

EX4

Evaluate: ∫₋∞^∞ (t² + 3t + 1) δ(t − 2) dt

Identify f(t) and the impulse location a

f(t) = t² + 3t + 1. The impulse δ(t − 2) fires at t = a = 2.

Apply sifting property: ∫f(t)δ(t−a)dt = f(a)

Result = f(2) = (2)² + 3(2) + 1 = 4 + 6 + 1 = 11.

System Properties

Linearity, time-invariance, causality, BIBO stability, memory, and invertibility. Q1 in every past exam — master your test procedures.

01

Linearity (Superposition)

Definition — two requirements must BOTH hold

Additivity (Part 1)T[x₁(t) + x₂(t)] = T[x₁(t)] + T[x₂(t)]
Homogeneity (Part 2)T[α·x(t)] = α·T[x(t)]

Combined: T[αx₁ + βx₂] = αT[x₁] + βT[x₂]. To test linearity: apply two arbitrary inputs, check if the response to their weighted sum equals the weighted sum of individual responses.

Nonlinear system

If either additivity or homogeneity fails, the system is nonlinear. Signs of nonlinearity: squared terms (x²), |x|, x·y, multiplication of inputs, initial conditions that don't scale linearly.

Test procedure: Let y₁ = T[x₁] and y₂ = T[x₂]. Compute T[αx₁ + βx₂]. If it equals αy₁ + βy₂, linear. Otherwise nonlinear.

02

Time-Invariance

Definition

If y(t) = T[x(t)], then T[x(t−t₀)] = y(t−t₀)

A time shift in the input causes the same time shift in the output — no more, no less. The system behaviour does not change over time.

Time-varying examples

y(t) = sin(t)·x(t−2) is time-varying — the multiplying factor sin(t) changes with time. y(t) = x(2t) is time-varying — the scaling 2t changes the relationship between input time and output time.

Test procedure: (1) Apply x(t−t₀) to system, get output A. (2) Shift original output y(t) by t₀ to get B = y(t−t₀). If A = B for all t₀, TI. If A ≠ B, time-varying.

03

LTI Systems

Linear Time-Invariant (LTI)

A system satisfying both linearity and time-invariance. The most important class in signal processing. Although real systems are often nonlinear and time-varying, LTI is a powerful and useful approximation near an operating point.

General LTI ODE formΣ aₖ y⁽ᵏ⁾(t) = Σ bₖ x⁽ᵏ⁾(t)
04

Causality

Definition

A system is causal if the output y(t) at any time t depends only on the input x(τ) for all τ ≤ t — present and past inputs only, never future inputs.

y(t) depends only on x(τ) for τ ≤ t

Causal ✓

y(t) = x(t) + x(t−1). Both x(t) (present) and x(t−1) (past) — causal. Any physical real-time system must be causal.

Non-causal ✗

y(t) = x(t+1). Uses a future input — non-causal (prophetic). Also: y(t) = x(−t) uses future values when t < 0.

05

BIBO Stability

Bounded Input Bounded Output

|x(t)| ≤ Mₓ < ∞ ⟹ |y(t)| ≤ Mᵧ < ∞

Every bounded input must produce a bounded output. For LTI systems: stable ⟺ ∫₋∞^∞ |h(τ)| dτ < ∞ (absolutely integrable impulse response).

Stable system ✓

y(t) = 0.5[x(t) − x(−t)]. If |x| ≤ M, then |y| = |0.5x(t) − 0.5x(−t)| ≤ M. Bounded. ✓

Unstable system ✗

y(t) = ∫₋∞ᵗ x(τ) dτ with input u(t). Output grows without bound → unbounded. ✗

06

Memory & Invertibility

Memory

Memoryless: y(t) = f(x(t)) only — output depends on present input only. Example: y(t) = x²(t).

With memory: integrators, differentiators, delay systems. The capacitor voltage depends on all past currents.

Invertibility

Invertible: distinct inputs always produce distinct outputs — we can recover x from y. y = 3x is invertible (x = y/3).

Non-invertible: y = x² maps both x and −x to the same output. Cannot recover x uniquely.

07

The Standard System Analysis Template

Use this structure for any Q1-type question (all exams)

Simplify the expression

Use sifting property of δ(t) to reduce ∫x(τ)[δ(t−τ)±δ(t+τ)]dτ to a time-domain expression.

State linearity: check T[αx₁+βx₂] = αy₁+βy₂

Apply the simplified expression to αx₁+βx₂ and verify superposition.

State time-invariance: check A vs B method

Shift input → get output A. Shift original output → get B. Compare.

State causality: does output use future inputs?

Look for x(t + something) or x(−t) terms when t > 0.

State BIBO stability: bound the output

Assume |x(t)| ≤ M. Show |y(t)| ≤ some finite multiple of M.

State memory: does output depend only on x(t)?

Any dependence on x at other times → system has memory.

Worked Examples

The exact style of Q1 and Q2 in every past exam. Master the 6-property analysis template.

EX1

Full analysis: y(t) = 0.5[x(t) − x(−t)] (the exam question)

Simplify (comes from the original integral form)

y(t) = x_o(t) — the odd part of x(t).

Linearity: compute T[αx₁ + βx₂]

T[αx₁+βx₂] = 0.5[(αx₁+βx₂) − (αx₁(−t)+βx₂(−t))] = α·0.5[x₁−x₁(−t)] + β·0.5[x₂−x₂(−t)] = αy₁+βy₂. Linear ✓

BIBO stability: assume |x(t)| ≤ M

|y(t)| = |0.5x(t) − 0.5x(−t)| ≤ 0.5|x(t)| + 0.5|x(−t)| ≤ 0.5M + 0.5M = M. BIBO stable ✓

Memory: does y(t) depend only on x(t)?

y(t) = 0.5x(t) − 0.5x(−t). The term x(−t) is x evaluated at −t ≠ t. Has memory ✗

Causality: does output use future inputs?

For t = −1: y(−1) needs x(+1) — a future value. Non-causal ✗

Time-invariance: apply x(t−t₀), compare outputs

Method A: T[x(t−t₀)] = 0.5[x(t−t₀) − x(−t−t₀)]. Method B: y(t−t₀) = 0.5[x(t−t₀) − x(−(t−t₀))] = 0.5[x(t−t₀) − x(t₀−t)]. A ≠ B since x(−t−t₀) ≠ x(t₀−t) in general. Time-varying ✗

EX2

Test y(t) = sin(t) · x(t − 2)

Time-invariance test

Apply x(t−t₀): y₁ = sin(t)·x(t−2−t₀). Shift y by t₀: y₂ = y(t−t₀) = sin(t−t₀)·x(t−t₀−2). y₁ ≠ y₂ (sin(t) vs sin(t−t₀)). Time-varying ✗

Linearity test

T[αx₁+βx₂] = sin(t)·(αx₁(t−2)+βx₂(t−2)) = α·sin(t)x₁(t−2) + β·sin(t)x₂(t−2) = αy₁+βy₂. Linear ✓

Causality

y(t) = sin(t)·x(t−2). At any time t, we use x(t−2) — a past value (delay of 2). Causal ✓

Conclusion: Linear, causal, but time-varying. The sin(t) multiplier changes over time — the system's gain varies.

EX3

Is y(t) = x²(t) linear?

Test with a counterexample

Let x₁(t) = 1, x₂(t) = 1, α = 1, β = 1. Then αx₁+βx₂ = 2.

T[αx₁+βx₂] = T[2] = 4

The output is 2² = 4.

αT[x₁] + βT[x₂] = T[1] + T[1] = 1 + 1 = 2

4 ≠ 2. Superposition fails. Nonlinear ✗

Squaring is the classic nonlinear operation. Any f(x) where the exponent isn't 1 (and there's no constant term violating zero-input → zero-output) will be nonlinear.

Convolution & LTI

The impulse response completely characterises an LTI system. The convolution integral is the fundamental input-output relationship — and the Laplace convolution theorem is exam-critical.

01

Impulse Response h(t)

The fundamental characterisation of LTI systems

For an LTI system, if we apply δ(t) as input, the output is h(t) — the impulse response. This single function completely describes the system for any input.

h(t) = T[δ(t)]

Why? Any input can be expressed as a weighted sum of shifted impulses. By LTI, the output is the same weighted sum of shifted impulse responses.

02

The Convolution Integral

Continuous-time convolution

Definitiony(t) = x(t) * h(t) = ∫₋∞^∞ x(τ) h(t−τ) dτ

4-step graphical procedure

Reflect h(τ) → h(−τ)

Fold h about the vertical axis to get its mirror image.

Shift → h(t−τ)

Slide the reflected h along the τ axis by amount t.

Multiply x(τ) · h(t−τ)

Find the overlap region and form the product.

Integrate the product

The area under the product is y(t) at this value of t.

03

Properties of Convolution

Commutative

x * h = h * x

Order doesn't matter — you can slide x over h or h over x.

Associative

(x * h₁) * h₂ = x * (h₁ * h₂)

Systems in series can be combined by convolving their impulse responses.

Distributive

x * (h₁ + h₂) = x*h₁ + x*h₂

Systems in parallel — their impulse responses add.

Sifting via convolution

x(t) * δ(t−t₀) = x(t−t₀)

Convolving with a shifted impulse just shifts the signal.

04

Step Response & Stability

Step response s(t)

s(t) = ∫₋∞ᵗ h(τ) dτ
h(t) = ds(t)/dt

Differentiate step response → impulse response. Integrate impulse response → step response.

BIBO via h(t)

BIBO stable ⟺ ∫₋∞^∞ |h(t)| dt < ∞

The impulse response must be absolutely integrable. An exponentially decaying h(t) is stable; a growing or undamped h(t) is not.

05

The theorem — appears in all past papers

StatementL{f₁(t) * f₂(t)} = F₁(s)·F₂(s)

Transfer function linkY(s) = H(s)·X(s) ←→ y(t) = h(t) * x(t)

Proof outline (know this for 15 marks)

Write L{f₁*f₂} as a double integral

L{∫₀ᵗ f₁(τ)f₂(t−τ)dτ} = ∫₀^∞ [∫₀ᵗ f₁(τ)f₂(t−τ)dτ] e^(−st) dt

Swap order of integration (τ from 0 to ∞, t from τ to ∞)

Substitute u = t − τ in inner integral

Laplace Transform

The Laplace transform converts differential equations into algebra. Region of convergence, poles, zeros, partial fractions, transfer functions, and block diagrams.

01

Definition & Why Laplace?

One-sided Laplace transform

DefinitionF(s) = L{f(t)} = ∫₀^∞ f(t) e^(−st) dt

s = σ + jω is complex. Laplace extends Fourier by multiplying by a decaying exponential e^(−σt) — this makes signals that don't have a Fourier transform (like unstable ones) converge.

Laplace advantage

  • Works on unstable signals (e^(at), a>0) — Fourier does not
  • Converts ODEs → algebraic equations in s
  • Initial conditions handled automatically
  • Transfer function H(s) captures system completely

Fourier as a special case

F(jω) = F(s)|_{s=jω}

Fourier transform is just the Laplace transform evaluated on the imaginary axis (σ = 0), when the ROC includes the jω axis.

02

Region of Convergence (ROC)

ROC — when does the integral converge?

For f(t) = e^(−at)u(t): L converges when Re(s) > −a. So the ROC is the right half-plane Re(s) > −a.

|e^(−st)| = e^(−σt) must → 0 as t → ∞, so σ > −a

The ROC is always a half-plane in the s-domain. The pole of F(s) is at s = −a, and the ROC is to the right of it.

Factor the denominator. Each root of the denominator is a pole.

Find residues using cover-up method

Apply inverse Laplace to each term

A/(s+a) → A·e^(−at)u(t)

Repeated poles

Transfer functions of parallel blocks add.

Negative feedback

G(s) = forward path, F(s) = feedback path. Most important for control systems.

Fourier Series

Any periodic function can be represented as a sum of sinusoids. Orthogonality, Fourier coefficients, compact form, and the signal spectrum.

01

Fourier's Key Observation

The synthesis idea

Fourier's insight: any periodic function can be represented as the weighted sum of a family of sinusoids. Euler had earlier shown sinusoids resolve into complex exponentials: Acos(ωt+φ) = Xe^(jωt) + X*e^(−jωt).

Why sinusoids? (from Dr. Khoo W9)For any linear system, a sinusoidal input → sinusoidal output (same frequency, different amplitude/phase)

Orthogonality — the mathematical foundation

Two functions are orthogonal if their inner product is zero. Sinusoids at different harmonic frequencies are orthogonal over one period:

∫₀ᵀ sin(nω₀t)cos(mω₀t) dt = 0 for all n, m
∫₀ᵀ e^(jnω₀t)·e^(−jmω₀t) dt = T·δ[n−m]

This orthogonality is why Fourier chose sinusoids — they don't "interfere" with each other, so each coefficient can be computed independently.

02

Fourier Series Representations

Trigonometric form

Synthesis (reconstruction)x(t) = a₀ + Σ[aₙcos(nω₀t) + bₙsin(nω₀t)]
Analysis (coefficients)a₀ = (1/T₀)∫x(t)dt (DC component / average value)
aₙ = (2/T₀)∫x(t)cos(nω₀t)dt
bₙ = (2/T₀)∫x(t)sin(nω₀t)dt

Complex exponential (compact) form

Synthesisx(t) = Σ cₙ e^(jnω₀t), n = −∞ to ∞
Analysiscₙ = (1/T₀) ∫[T₀] x(t) e^(−jnω₀t) dt

c₀ = a₀ (DC/average). For real signals: c₋ₙ = cₙ* (complex conjugate symmetry).

03

Even/Odd Signals & Fourier

Even signal → cosine series

If x(t) is even: bₙ = 0 for all n. Series has only cosine terms (and DC). This is a cosine series.

Odd signal → sine series

If x(t) is odd: a₀ = 0 and aₙ = 0 for all n. Series has only sine terms. This is a sine series.

04

Spectrum & Applications

Signal spectrum

The compact Fourier series allows a periodic signal to be plotted as a spectrum — a plot of amplitude |cₙ| and phase ∠cₙ vs frequency nω₀. This is the bridge to frequency-domain analysis.

Laplace vs Fourier — when to use which?

  • Laplace: transient analysis, control theory, initial conditions, unstable signals, solving ODEs
  • Fourier: steady-state analysis, frequency response, communications, filtering, bandwidth
  • In communications and control, both are often needed

Properties of Fourier series

  • Linearity: F{αx+βy} = αX+βY
  • Time shift: x(t−t₀) ↔ cₙe^(−jnω₀t₀)
  • Parseval: P = Σ|cₙ|²
  • Conjugate: real x(t) → c₋ₙ = cₙ*

Second-order systems

General 2nd order TFH(s) = ωₙ²/(s²+2ζωₙs+ωₙ²)

ζ>1: overdamped (distinct real poles), ζ=1: critically damped (repeated pole), ζ<1: underdamped (complex poles → oscillation).

Find the Fourier series of a square wave

Compute bₙ = (2/T₀)∫₀^T₀ x(t)sin(nω₀t)dt

Z-Transform

The discrete-time counterpart of the Laplace transform. Converts difference equations into algebra, enables discrete system analysis, and defines discrete BIBO stability via the unit circle.

01

Why Discrete-Time?

From continuous to discrete

A continuous signal x(t) sampled at period T gives a discrete sequence x[n] = x(nT). Modern engineering uses microcontrollers to implement filters digitally — continuous-time analysis provides the foundation, then we "migrate" to discrete time.

Sampling (discretization)x[n] = x(nT) = x(t)|_{t=nT}

Discrete difference equations

1st ordery[n] − ay[n−1] = x[n]
General Nth orderΣ aₖy[n−k] = Σ bₖx[n−k]

Discrete convolution

y[n] = x[n] * h[n] = Σ x[k]h[n−k]

Discrete LTI also characterised completely by its unit pulse response h[n].

02

The Z-Transform

Definition

One-sided Z-transformX(z) = Σ x[n] z^(−n), n = 0 to ∞

Derived by Laplace-transforming the sampled signal and substituting z = e^(sT). The complex variable z = e^(sT) maps the s-plane to the z-plane.

s-plane to z-plane mapping

z = e^(sT) = e^(σT)e^(jωT)
  • Left-half s-plane (stable continuous) → inside unit circle |z| < 1
  • Imaginary axis jω → unit circle |z| = 1
  • Right-half s-plane (unstable) → outside unit circle |z| > 1

Procedure — mirrors Laplace for ODEs

Take Z-transform of both sides

Use shifting property: Z{y[n−k]} = z^(−k)Y(z) + IC terms.

Stability condition

Discrete BIBO stable ⟺ Σ|h[n]| < ∞ ⟺ all poles of H(z) inside unit circle: |pₖ| < 1

Just as continuous stability requires poles in the left-half s-plane, discrete stability requires poles strictly inside the unit circle in the z-plane.

Take Z-transform using shifting property

Q1 — Even/Odd orthogonality proofs
(a) Prove: ∫₋∞^∞ x_e(t)·x_o(t) dt = 0 (15 marks)
(b) Show: ∫₋∞^∞ x(t) dt = ∫₋∞^∞ x_e(t) dt (10 marks)

Part (a) — Proof that ∫x_e · x_o dt = 0

Let I = ∫₋∞^∞ x_e(t)·x_o(t) dt. Substitute t = −τ (dt = −dτ), flipping limits back:

I = ∫₋∞^∞ x_e(−τ)·x_o(−τ) dτ = ∫₋∞^∞ x_e(τ)·(−x_o(τ)) dτ = −I

So I = −I, meaning 2I = 0, therefore I = 0. QED ✓

Part (b) — Proof that ∫x dt = ∫x_e dt

Since x(t) = x_e(t) + x_o(t): ∫x dt = ∫x_e dt + ∫x_o dt. We need ∫x_o dt = 0.

Substitute t = −τ in ∫₋∞^0 x_o(t)dt → ∫₀^∞ x_o(−τ)dτ = −∫₀^∞ x_o(τ)dτ. These cancel. So ∫x_o dt = 0.

Therefore ∫x dt = ∫x_e dt + 0 = ∫x_e dt. QED ✓

Q2/Q5 — System analysis: y(t) = 0.5∫x(τ)[δ(t−τ)−δ(t+τ)]dτ
(a) What does this system do?   (b) BIBO stable?   (c) Linear?   (d) Memoryless?   (e) Causal?   (f) Time-variant?

Step 1 — Simplify using sifting property

∫x(τ)δ(t−τ)dτ = x(t) and ∫x(τ)δ(t+τ)dτ = x(−t). Therefore:

y(t) = 0.5[x(t) − x(−t)] = x_o(t) (the ODD part of x)

(a) The system extracts the ODD part of the input signal.

(b) BIBO stable? YES. |y| = |0.5x(t)−0.5x(−t)| ≤ 0.5|x(t)|+0.5|x(−t)| ≤ M. ✓

(c) Linear? YES. T[αx₁+βx₂] = 0.5[(αx₁+βx₂)−(αx₁+βx₂)|_{reflected}] = αT[x₁]+βT[x₂]. ✓

(d) Memoryless? NO. y(t) depends on x(−t), which is a value at a different time instant. ✗

(e) Causal? NO. For t<0, the term x(−t) = x(|t|) uses a future value of x. Non-causal. ✗

(f) Time-variant? YES. Shift input by t₀: output becomes 0.5[x(t−t₀)−x(−t−t₀)]. But shifting y by t₀ gives 0.5[x(t−t₀)−x(t₀−t)]. Since x(−t−t₀) ≠ x(t₀−t) in general → time-varying. ✗

Q5 — Prove the Laplace convolution theorem
Prove: If L{f₁(t)} = F₁(s) and L{f₂(t)} = F₂(s), then L{f₁(t)*f₂(t)} = F₁(s)F₂(s)

Proof of convolution theorem

Step 1: Write L{f₁*f₂} = ∫₀^∞ [∫₀ᵗ f₁(τ)f₂(t−τ)dτ] e^(−st) dt

Step 2: Swap integration order (τ: 0→∞, t: τ→∞):

= ∫₀^∞ f₁(τ) [∫_τ^∞ f₂(t−τ)e^(−st) dt] dτ

Step 3: Substitute u = t−τ in inner integral (dt=du, lower limit 0):

Inner = e^(−sτ) ∫₀^∞ f₂(u)e^(−su) du = e^(−sτ)F₂(s)

Step 4: Substitute back: = F₂(s)·∫₀^∞ f₁(τ)e^(−sτ)dτ = F₁(s)·F₂(s) QED ✓

Part (b) — Capacitor impulse response (C = 1F)

For a capacitor: i(t) = C dv/dt → v(t) = (1/C)∫₋∞ᵗ i(τ)dτ. With C=1F, x=i, y=v: this is an integrator.

Apply δ(t) → h(t) = ∫₋∞ᵗ δ(τ)dτ = u(t). Impulse response: h(t) = u(t).

Step response via convolution: s(t) = u(t)*u(t) = ∫₀ᵗ 1·1 dτ = t for t≥0. Step response: s(t) = t·u(t).
